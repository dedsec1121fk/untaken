# Changelog

## v4.0.0 (Pro CLI upgrade)

- Added parallel workers (`--workers`)
- Added resume mode (`--resume`)
- Added per-username cache (`--cache-dir`, `--cache-ttl`)
- Added NDJSON export and summary JSON output
- Added profiles (`fast`, `balanced`, `conservative`)
- Added proxy / connect-timeout / TLS-insecure options
- Added rotated user-agent pool support
- Added input shuffle option
- Added output print formats (`pretty`, `tsv`, `jsonl`)
- Added optional raw response body capture directory
- Added config file support (`--config`)
- Added interrupt-safe summary writing
- Improved docs and usage examples

## v3.x

- Validation, strict mode, retries/timeouts, CSV/JSON exports, summary files, anti-bot detection
